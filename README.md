# Portfolio Tracker

A command-line portfolio tracker built in Python.

## Installation

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt

## How to Run

python main.py

## Features

- Add assets with ticker, sector, asset class, quantity and purchase price
- View portfolio table with current prices, transaction value, current value and gain/loss
- Plot historical price charts for individual or all tickers
- View portfolio weights broken down by asset and by sector/asset class
- Run a Monte Carlo simulation over 15 years with 10,000 simulated paths

## Usage

On startup you will see a menu:

0. Load test portfolio  
1. Add asset  
2. View portfolio  
3. Plot prices  
4. View weights breakdown  
5. Run simulation  
6. Quit

Select 0 to load a sample portfolio and explore all features immediately.
